# handover.md · Session Handover Log (Team Convention)

> This file is a **team convention**, not an Antigravity product feature. At the end of a
> significant working session, ask the agent: *"Update handover.md with what was completed,
> decisions made, open issues, and the next step."* The next session (or the next person)
> starts by reading it.

## How to use
- Newest entry on top. Keep each entry short: done / decided / open / next.
- Never include salary figures or personal identifiers in this file, it travels with the project.

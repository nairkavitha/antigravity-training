# context.md · Standing Brief for the Agent (Team Convention)

> This file is a **team convention**, not an Antigravity product feature. We keep it in the
> workspace so any agent (or new colleague) can read the standing rules of this project in
> one place. Reference it explicitly in your prompts, e.g. "follow the rules in context.md".

## Who we are
- Org: Reliance Jio, Corporate HR training sandbox (HR_Mission_Control).
- Users: HR professionals learning agentic AI orchestration. Non-developers.
- Audience for outputs: HR managers and business leaders. Professional, concise, Indian English.

## Standing rules (apply to every task)
1. **Synthetic data only.** Everything in this folder is fictional. Never request or use real employee data.
2. **Privacy first.** Do not include salary figures, emails or any personal identifiers in outputs unless the task explicitly requires them, and never in public-facing summaries.
3. **Stay in the sandbox.** Read and write only within the HR_Mission_Control folder. Save all generated outputs under `exercises/`.
4. **Cite your sources.** Every figure in an output should be traceable to a file in `data/` or `policies/`. If a value is estimated, label it `[ESTIMATED]`.
5. **Plan before you act.** For multi-step tasks, present an implementation plan and wait for approval before writing files.
6. **Currency and dates.** Amounts in INR. Fiscal year runs April–March. Today's training cohort works on FY 2026-27 scenarios.

## Key terms
- **PL/CL/SL:** Privilege / Casual / Sick Leave (see leave policy).
- **Compa-ratio:** current salary ÷ market midpoint for the employee's grade.
- **HITL:** Human in the loop, a human reviews plans and outputs before they are used.

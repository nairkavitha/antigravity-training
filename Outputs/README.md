# Antigravity 2.0 Training & Upskilling Program

A single-file interactive training portal for upskilling HR professionals in Agentic AI Orchestration at Reliance Jio.

---

## 📁 Folder Structure

```
outputs/
├── index.html              # Main training portal — open this in a browser
├── HR_Mission_Control/     # Synthetic sandbox data for hands-on exercises
│   ├── data/               # employees.csv and other datasets
│   ├── policies/           # Sample HR policy documents
│   └── templates/          # Document templates
├── README.md               # This file
└── archived/               # Older standalone files and planning docs
```

---

## 🚀 Getting Started

Open `index.html` in any modern browser. No server or installation needed.

The portal covers 5 modules:
1. **Foundations** — What agentic AI is and how it thinks
2. **Setup & Tour** — Antigravity 2.0 interface walkthrough
3. **Core Skills** — Goal-writing, tool use, and subagents
4. **Build** — Hands-on project using HR_Mission_Control sandbox data
5. **Help & Glossary** — Troubleshooting, prompts, and AI terminology flip cards

---

## 🗂 Sandbox Data

All exercises use synthetic data in `HR_Mission_Control/`. No real employee PII is included. Do not connect to production systems (Workday, SAP, HRMS) without explicit approval.
